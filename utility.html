func_elapsed

func_elapsed函数
提供函数执行计时功能，可用于自动化测试或者性能调优。支持任意函数形式和任意入参类型。
使用方法如下：
example1 普通函数

void func(size_t x) {
size_t sum{0};
for (const auto& i : irange(x)) {
sum += i;
}
}
std::cout << "func elapsed " << dsp::func_elapsed(&func, 10000) << "ms" << std::endl;
