公式表达式

基本概念

公式表达式是对给定的字符串进行公式语法分析后得到的表达式对象。
表达式由几种元素组成：

  • 常数，统一解析为double类型，支持科学计数法形式的输入
  • 变量，必须以英文字母开头，由英文字母、数字和下划线组成
  • 操作符，分为一元操作符、二元操作符、函数操作符

以“-sin(x + 1.)”为例说明，"-"解析为一元操作符，"sin"解析为函数操作符，“x”解析为变量， "+"解析为二元操作符，"1."解析为常数。

使用方法如下:

#include <dsp/utility/formula_expression.hpp>
std::string str{“x * (x + y - 1)”};
dsp::expression::Formula f;
auto ok = dsp::expression::parse(str, f);
if (ok) {
f.set("x", 1.);
f.set("y", 0.5);
std::cout << "Result: " << dsp::expression::eval(f) << "\n";
}

支持范围

一元操作符支持范围如下表：

操作符 说明
+ 正号
- 负号
! 取反
~ 按位取反

二元操作符支持范围如下表:

操作符 分类 说明
+ 算数运算符 加法
- 减法
* 乘法
/ 除法
% 取模
== 关系运算符 判断相等
!= 判断不等
> 大于
< 小于
>= 大于等于
<= 小于等于
&& 逻辑运算符 与
&#124; &#124; 或

|&|位运算符按位与| |&#124;|^|按位或| |^|^|按位异或| |<<|^|左移| |>>|^|右移|

函数操作符支持范围如下表:

操作符 说明
cond 实现三目运算符(?:)的功能，输入3个参数
abs 取绝对值
max 取最大值，可输入任意数量的参数
min 取最小值，可输入任意数量的参数
ceil 向上取整
floor 向下取整
sqrt 求平方根
pow 幂运算
sin 正弦函数
cos 余弦函数
tan 正切函数
