详细描述

简单的数据序列化/反序列化类。常用于构造内部网络报文的data。可以使用pop/push函数或者使用>>/<<操作符来进行数据存取，数据先进先出 且连续存放，可以通过data() 获取数据首地址，通过size() 获取数据总长度。自定义类型可以通过声明友元函数来支持memory_io的操作，下面是一个示例：

struct foo {
foo(int _x = 0, int _y = 0, float _z = 0.) : x(_x), y(_y), z(_z) {}
friend memory_io &operator<<(memory_io &, const foo &);
friend memory_io &operator>>(memory_io &, foo &);
friend std::ostream &operator<<(std::ostream &, const foo &);
public:
int x{0};
int y{0};
float z{0};
};
inline memory_io &operator<<(memory_io &mio, const foo &f) {
mio << f.x << f.y << f.z;
return mio;
}
inline memory_io &operator>>(memory_io &mio, foo &f) {
mio >> f.x >> f.y >> f.z;
return mio;
}
inline std::ostream &operator<<(std::ostream &os, const foo &f) { os << f.x << " " << f.y << " " << f.z; }
memory_io mio{};
foo f1{1, 2, 3.5};
foo f2{};
mio << f1;
mio >> f2;
std::cout << f2 << std::endl;
// Output: 1 2 3.5
