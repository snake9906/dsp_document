dsp::func_elapsed

提供函数执行计时功能，可用于自动化测试或者性能调优。支持任意函数形式和任意入参类型。
使用方法如下：

普通函数

void func(size_t x) {
size_t sum{0};
for (const auto& i : irange(x)) {
sum += i;
}
}
std::cout << "func elapsed " << dsp::func_elapsed(&func, 10000) << "ms" << std::endl;

成员函数

class Foo {
public:
void func(size_t x) {//...}
}
Foo f;
std::cout << "func elapsed " << dsp::func_elapsed(&Foo::func, f, 10000) << "ms" << std::endl;

lambda函数

std::cout << func elapsed " << dsp::func_elapsed([](int x){//...}, 10000) << "ms" << std::endl;

std::function

std::function<void(int)> func = [](int x){//...};
std::cout << func elapsed " << dsp::func_elapsed(func, 10000) << "ms" << std::endl;

dsp::sleep

sleep函数，可选普通精度和高精度，其中普通精度的误差范围在10几毫秒左右，高精度的误差范围在几十微秒左右
当sleep超过1秒时，强制选择普通精度。

// sleep 100ms 普通精度
std::cout << "sleep for " << dsp::func_avg_elapsed(10, dsp::sleep, 100, false) << " ms" << std::endl;
// Output: 111.416 ms
// sleep 100ms 高精度
std::cout << "sleep for " << dsp::func_avg_elapsed(10, dsp::sleep, 100, true) << " ms" << std::endl;
// Output: 100.008 ms
